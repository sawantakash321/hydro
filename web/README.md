This is the front end of the Hydro box dex project. It's developed using react. The project is created by the [create-react-app](https://github.com/facebook/create-react-app) command.

## How to develop

### Step 0: Install dependences

	npm install

### Step 1: Start server

	npm run start

## How to build

### Step 0: Install dependences

	npm install

### Step 1: Build

	npm run build
